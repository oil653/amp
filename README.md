# Purpose
AMP is most of a personal project. I could never find fit in any of the music streaming apps, and i tried so many. Most have small issues, others have bigger ones.  

# Platforms
I'm planning to support for Linux, Windows and Android, and later for the web as well. The music player backend allows playback on iOS, MacOs, but I never used an apple device, and not planning to do so either. I will NOT add support for these platforms, BUT if you want to add support and follow the patterns that are in the rest of the code you're free to add support for those platforms.

I'm developing AMP on Arch Linux, so dont be surprised if some non release version doesnt work on all platforms.
